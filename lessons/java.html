Object-Oriented Languages
Java is best known as one of the first popular object-oriented programming languages. This means that the entire language focuses on entities called "objects," whihc are basically collections of data associated with a single entity. Java is very structured in its object-oriented design, so it's a good language for new learners who like rules (kind of like training wheels). I'll add a detailed breakdown on Java's syntax later. For now, just refer to the code I wrote in class today.
